# 未來街坊圈 (Future Neighborhood Circle) 用戶手冊

歡迎使用 **未來街坊圈**！這是一個專為社區打造的互動平台，讓您輕鬆參與社群活動、分享生活動態，並與鄰居保持緊密聯繫。

---

## 1. 快速入門

### 1.1 註冊帳號
如果您是第一次使用，請按照以下步驟註冊：
1. 點擊首頁的 **「註冊」** 按鈕。
2. 輸入您的 **電子郵件** (作為登入帳號)。
3. 設定您的 **使用者名稱** 與 **密碼**。
4. 點擊確認註冊，系統將自動為您登入。

### 1.2 登入系統
1. 在首頁輸入您的 **電子郵件** 或 **手機號碼**。
2. 輸入密碼後點擊 **「登入」**。

> **小提示**：系統支援自動登入，下次開啟時無需重新輸入密碼。

---

## 2. 社群互動功能

### 2.1 瀏覽動態牆
登入後，您會看到社群的動態牆（Feed）：
*   **最新消息**：依照時間排序，顯示所有成員的貼文。
*   **置頂公告**：帶有 📌 標記的貼文是管理員發佈的重要公告，請務必留意。

### 2.2 發佈貼文
想分享生活點滴嗎？
1. 點擊畫面下方的 **「+」** 或 **「發佈」** 按鈕。
2. 輸入文字內容。
3. (選用) 上傳圖片或影片。
4. 點擊 **「送出」**，您的貼文就會即時顯示在動態牆上。

### 2.3 互動 (點讚與評論)
*   **點讚**：點擊貼文下方的 ❤️ 愛心圖示，表達您的喜愛。
*   **評論**：點擊 💬 對話框圖示，輸入留言與鄰居互動。

---

## 3. 活動與報名

### 3.1 瀏覽活動
在 **「活動」** 分頁中，您可以查看近期即將舉辦的所有活動：
*   顯示活動時間、地點、費用（或免費）。
*   點擊活動卡片可查看詳細介紹。

### 3.2 線上報名
1. 進入感興趣的活動頁面。
2. 點擊 **「立即報名」**。
3. 系統會顯示報名成功訊息。
4. 若活動需要付費，請依照指示完成繳費（目前支援線下繳費或轉帳確認）。

### 3.3 查看我的活動
在 **「個人中心」** -> **「我的活動」** 中，您可以查看所有已報名的活動狀態。

---

## 4. 個人中心

### 4.1 編輯個人資料
1. 點擊右下角的 **「我的」** 圖示。
2. 點擊頭像或 **「編輯資料」**。
3. 您可以修改：
    *   更換頭像 (Avatar)
    *   更新個人簡介 (Bio)
    *   修改暱稱

### 4.2 查看會籍資訊
在個人中心，您可以看到您的：
*   **會員編號**
*   **會員等級**
*   **加入時間**

---

## 5. 管理員專區 (僅限管理員)

> 此區域功能僅對擁有 Admin 權限的帳號開放。

### 5.1 發佈公告
1. 在發文介面中，勾選 **「設為公告」** 或 **「置頂」**。
2. 發佈後，該訊息將永遠顯示在動態牆最上方。

### 5.2 建立活動
1. 進入活動頁面，點擊 **「建立活動」**。
2. 填寫活動資訊：
    *   標題與描述
    *   開始與結束時間
    *   地點與名額限制
    *   費用 (若為免費請填 0)
    *   早鳥優惠 (選填)
3. 確認發佈。

### 5.3 成員管理
1. 進入 **「社群管理」** 後台。
2. 您可以查看所有會員列表。
3. 操作選項：
    *   **調整等級**：升級或降級會員。
    *   **停權**：若會員違反社群守則，可暫時停用其帳號。

### 5.4 數據儀表板
在首頁的管理員入口，可查看：
*   總會員數 / 活躍會員數
*   本月活動場次
*   總營收概況
