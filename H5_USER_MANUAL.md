# 未來街坊 H5 行動應用程式用戶手冊

## 1. 簡介 (Introduction)

**未來街坊** 是一款專為社區成員和管理員設計的行動應用程式 (H5 App)。透過此應用程式，會員可以輕鬆瀏覽社區活動、管理個人會籍，而管理員則可以高效地進行活動管理和現場簽到。

### 如何訪問
本應用程式為 H5 網頁應用，您可以透過手機瀏覽器直接訪問應用程式網址。建議將網址加入主畫面以獲得類似原生 App 的體驗。

---

## 2. 登入與認證 (Login & Authentication)

### 登入步驟
1. 打開應用程式，您將看到登入頁面。
2. 輸入您的帳號 (電子郵件或手機號碼) 和密碼。
3. 點擊「登入」按鈕。

### 帳號權限與等級
系統設有四種會員等級，分別對應不同的權限與識別：

1.  **系統管理員 (System Administrator)** 👑
    - 最高權限，可進入管理後台。
    - 負責管理會員、活動、繳費及系統設定。
    - 可使用掃碼簽到功能。

2.  **未來街坊圈圈委 (Circle Committee)** ⭐⭐⭐
    - 社區的核心成員，協助推動社區事務。
    - 擁有專屬識別徽章。

3.  **未來街坊圈圈民 (Circle Citizen)** ⭐⭐
    - 正式認證的社區居民。
    - 可報名參加會員專屬活動。
    - 享有社區投票或特定福利（視社區規劃而定）。

4.  **未來街坊圈圈友 (Circle Friend)** ⭐
    - 剛加入社區的訪客或體驗成員。
    - 可瀏覽公開資訊及參加公開活動。

## 3. 會員功能指南 (Member Features)

登入後，底部導航欄提供以下主要功能：

### 3.1 首頁 (Home)
- **儀表板概覽**：顯示您的會員狀態摘要。
- **快速入口**：提供常用功能的快速連結。
- **最新消息**：顯示社區的最新公告或活動推薦。

### 3.2 活動 (Events)
在此頁面，您可以瀏覽所有社區活動。
- **活動分類**：透過上方標籤切換「全部」、「即將舉行」或「已結束」的活動。
- **活動詳情**：點擊任一活動卡片，可查看詳細資訊，包括：
  - 時間與地點
  - 費用 (包含早鳥優惠資訊)
  - 名額限制
  - 活動說明
- **報名活動**：
  - **免費活動**：點擊報名後確認即可。
  - **付費活動**：選擇付款方式 (如銀行轉賬或現金)，提交報名後需完成繳費。

### 3.3 我的會籍 (Membership)
- 查看您的電子會員卡。
- 顯示會員編號、等級及有效期。
- 參加活動時，可出示此頁面的 QR Code 供管理員掃描簽到。

### 3.4 個人檔案 (Profile) / 我的
- **個人資料**：查看或編輯您的基本資料。
- **帳號設定**：修改密碼或登出帳號。

## 4. 管理員功能指南 (Admin Features)

管理員擁有額外的權限來維護社區運作。

### 4.1 儀表板 (Dashboard)
- 查看系統運作概況，如會員總數、近期活動數據等。

### 4.2 掃碼簽到 (Scanner)
用於活動現場簽到。
- **進入掃描器**：點擊管理選單中的「掃碼簽到」。
- **選擇活動**：在掃描頁面下方，務必先選擇「當前活動」。
- **掃描 QR Code**：
  - 允許瀏覽器使用相機權限。
  - 將會員出示的 QR Code 對準掃描框。
  - 系統將自動辨識並完成簽到，顯示綠色成功圖示。
- **手動簽到**：若相機無法使用，可在下方輸入框手動輸入「會員編號」進行簽到。

### 4.3 會員管理 (Member Management)
- **會員列表**：瀏覽所有註冊會員。
- **會員詳情**：點擊會員可查看詳細資料、繳費記錄及參與活動歷史。

### 4.4 活動管理 (Event Management)
- **新增活動**：在活動頁面點擊右下角的「+」按鈕 (僅管理員可見)。
- **編輯活動**：在活動詳情頁面，點擊標題旁的編輯圖示。
- **管理內容**：設定活動標題、時間、地點、費用、早鳥優惠及名額限制。

### 4.5 繳費管理 (Payments)
- 查看會員的繳費申請。
- 審核銀行轉賬證明，確認收款後更新繳費狀態。

## 5. 常見問題 (FAQ)

### Q1: 無法開啟掃描器相機？
**A:** 請確保您的瀏覽器已獲得相機權限。
- **iOS (Safari)**：請在網址列點擊「大小」圖示 -> 網站設定 -> 相機 -> 允許。
- **Android (Chrome)**：點擊網址列鎖頭圖示 -> 權限 -> 相機 -> 允許。
- 若仍無法使用，請嘗試重新整理頁面或使用「手動輸入會員編號」功能。

### Q2: 忘記密碼怎麼辦？
**A:** 請聯繫社區管理員協助重置密碼。

### Q3: 報名後如何取消？
**A:** 目前 App 暫不支援自行取消報名，請直接聯繫管理員進行處理。
